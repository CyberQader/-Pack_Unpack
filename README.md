# 🛠️ Dynamic Skills Tracker

سكربت بايثون احترافي يوضح قوة الـ **Packing** في الدوال، حيث يسمح للمستخدم بإدخال عدد غير محدود من المهارات دون تعديل الكود.

## 🚀 المميزات (Features)
* **Variable Arguments (`*args`):** استخدام النجمة `*` لاستقبال أي عدد من المهارات (Flexible Input).
* **Data Iteration:** معالجة المهارات المدخلة باستخدام `for loop`.
* **String Formatting:** تحويل كافة النصوص لـ `UPPERCASE` لعرض احترافي.

## 📋 مثال على التشغيل (Execution)
عند استدعاء الدالة بهذا الشكل:
`show_detils('abood','Html','Js','Python')`
